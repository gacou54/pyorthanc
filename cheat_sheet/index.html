    Cheat sheet

    Orthanc has a very useful cheat sheet for finding functionalities and REST API routes. This page contains the mappings between REST API paths and methods in the pyorthanc.Orthanc class.

    Note that pyorthanc.Orthanc is the low level client with which users can interact with Orthanc. Higher level functionalities such as Patient, Study, Series, Instance, find_patients/studies/series/instances, or Modality might be more convenient to use.
